Code Editor

I used Notepad++ code editor because of its simplicity. There wasn't a huge learning curve, so I was able to get up and running quickly. Some great features are syntax highlighting, auto-completion, code hints, themes, and plug-ins for both the program itself and for different programming languages. You can even tweak the program to function with how you like to code.
