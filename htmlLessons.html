HTML 5

Mozilla Developer Network

Since I am well-versed with HTML already, I only used a reference page for HTML 5. MDN's site lists visual representations of what each element looks like when used. I keep this open when coding just in case. By clicking an element's link, the site allows you to work with the element interactively, and see it used in both HTML5 and CSS(3) code. It even helps with accessibility by listing the browsers where the element can be used, and how similar code will produce the same result. The most important feature of this site DOM feature to move beyond basic HTML and CSS and go into object-oriented programming.
